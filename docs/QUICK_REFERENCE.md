# 3000METER.com - Quick Reference

## Prosjekt Oversikt

3000METER.com er en profesjonell tempo kalkulator for baneløpere, bygget med moderne web teknologier og fokusert på brukervennlighet og ytelse.

## Rask Start

1. Klon repositoriet
2. Åpne `index.html` i nettleseren eller start lokal server
3. Velg distanse, angi måltid/tempo, og klikk "Beregn"

## Teknologi

- **HTML5** - Semantisk markup
- **CSS3** - Moderne styling med Space Grotesk font
- **Vanilla JavaScript** - Ingen rammeverk
- **SVG** - Bane visualisering
- **Chart.js** - Tempo grafikk
- **PWA** - Progressive Web App støtte

## Viktige Filer

- `index.html` - Hoved HTML struktur
- `js/script.js` - All JavaScript logikk
- `css/styles.css` - Alle stiler
- `pwa/manifest.json` - PWA konfigurasjon
- `pwa/sw.js` - Service Worker

## Dokumentasjon

Se `docs/` mappen for utfyllende dokumentasjon:
- Features, Architecture, Development, API

## Struktur

Prosjektet er organisert i logiske mapper:
- `css/` - Stilark
- `js/` - JavaScript
- `assets/` - Statiske ressurser
- `pwa/` - PWA filer
- `docs/` - Dokumentasjon

## Underhåll

- Koden er organisert i logiske seksjoner i `script.js`
- CSS følger BEM metodikk hvor mulig
- Kommentarer dokumenterer komplekse funksjoner
- Ingen build tools nødvendig - ren HTML/CSS/JS
